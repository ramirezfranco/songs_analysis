Identifying Names Entities in reggaeton lyrics¶

In this notebook, I find subsequences of words identified as criminal charges and causes of death in reggaeton lyrics.

Important note: most of the reggaeton lyrics are written in Spanish¶

  1. First, I read the JSON file generated in step 2 in the "reggaeton_corpus" jupyter notebook and produced a list of lists where every record is a reggaeton song, as we did in step 3 in the same notebook.
In [13]:
with open('regaetton_songs.json', 'r') as file:
    corpus = json.load(file)
In [3]:
documents = [d['lyrics'] for artist in corpus.values() if artist for d in artist.values()]
  1. Then, I run the Stanford CoreNLP server (see details in https://stanfordnlp.github.io/CoreNLP/corenlp-server.html) and get the Name Entity Recognizer annotations for all the corpus. The "ners" variable produced is a dictionary where every key is a class of entities, and every value is a list of subsequences

ners  = util.ners_in_corpus(documents)
